# Contributing to Saymore

Issues, design discussions, documentation feedback, and reproducible bug reports
are welcome.

Saymore uses a source-available license and intends to offer commercial licenses
and services. External code contributions therefore require an explicit
Contributor License Agreement (CLA) that gives the project the rights needed to
use and commercially license the contribution while the contributor retains
copyright.

The CLA signing workflow is not yet available. Until it is published, please
open an issue before writing code; maintainers will not merge external code or
documentation contributions that contain copyrightable implementation content.
This avoids accepting work under unclear or insufficient rights.

By participating in project spaces, contributors agree to communicate
professionally and keep reports free of secrets, credentials, private user data,
and proprietary material they do not have permission to share.
