# 采用 MIT 或 Apache-2.0 双许可

状态：已由
[`0005-use-polyform-shield-source-available-license.md`](0005-use-polyform-shield-source-available-license.md)
取代。

Saymore 客户端和核心库采用 `MIT OR Apache-2.0` 双许可，以最大化 Rust
生态兼容性、个人和企业采用，并通过 Apache-2.0
提供明确专利授权。我们接受第三方可以创建闭源商业分支这一代价；未来托管服务可以独立运营，不通过限制开源客户端许可证实现商业保护。

## Considered Options

AGPL
可以要求网络服务公开修改，但会提高企业采用和集成阻力，不符合当前优先扩大开源客户端生态的目标。
